# Jupyter Notebooks

This folder contains the Jupyter notebooks used for the technical demonstration.

## Notebooks

- `mono-datalake-hadoop-spark.ipynb`  
  Implements a traditional data lake pipeline using Hadoop HDFS and Apache Spark without transactional guarantees.

- `lakehouse-delta-lake.ipynb`  
  Implements a lakehouse pipeline using Delta Lake on Apache Spark with ACID-compliant operations.

The notebooks are designed to be executed independently for comparison purposes.
